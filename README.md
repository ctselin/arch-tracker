Decision Tracker for Architecture Projects
A lightweight, open-source project management tool built specifically for architecture design teams. No installation, no account, no subscription — just open the HTML file in any browser and start tracking.
---
Why This Exists
Architecture projects generate a constant stream of decisions, client comments, and consultant coordination items across emails, meetings, redlines, and phone calls. These get scattered across Excel sheets, inboxes, and people's memories.
This tool gives your team one place to log every decision, track follow-ups, and make sure nothing falls through the cracks — especially when a new team member joins mid-project.
---
Features
Decision Log — Log every owner/consultant comment with Original Comment, Action Required, source, and format
Follow-up Tracking — Set chase dates; overdue items surface automatically in the Chase Bar
Sub-tasks — Each entry can have follow-up tasks with their own status and chase dates
Project Dashboard — Team directory, design fee tracker, key document links, project info
Milestone Schedule — Timeline view with weekday labels, color-coded by urgency
Export to Calendar — Export milestones as `.ics` for Outlook, Google Calendar, Apple Calendar
Print Report — One-click print of all active items for meetings
Save / Load — Data saved as a `.json` file; load it on any computer or share with teammates
Sort & Filter — Arrange by received date, status, source, or assigned person
---
How to Use
Option 1 — Use directly in browser
Open https://ctselin.github.io/arch-tracker in any browser. No installation needed.
Option 2 — Download and run locally
Download `index.html`
Open it in any browser (Chrome, Edge, Safari, Firefox)
Done
Saving your data
Click ↓ Save Data to download a `.json` file
Store it in OneDrive, Dropbox, or any shared folder
On another computer, open the tool and click ↑ Load File to restore your data
Share the `.json` with teammates so they can load the same project
---
Data & Privacy
All data is stored locally in your browser or in the `.json` file you save. Nothing is sent to any server. This tool has no backend, no database, no analytics.
